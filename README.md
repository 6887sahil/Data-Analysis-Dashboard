## Bike Sales Dashboard - Excel Project
## 📊 Project Overview
This project is an interactive Bike Sales Dashboard created using Microsoft Excel.  
The dashboard analyzes customer data to understand factors affecting bike purchases.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## 📈 Dashboard Insights
- Average Income Per Purchase
- Customer Age Brackets
- Customer Commute Distance
- Regional and Education Filters

## 📂 Project Files
- Raw Dataset
- Working Sheet
- Pivot Tables
- Interactive Dashboard
